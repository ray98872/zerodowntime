Zero-Downtime Blue-Green Deployment

Updating a live GenAI Copilot's RAG index without dropping a single user request. A new Green revision is deployed alongside the live Blue revision, health-verified on its private URL, then traffic is shifted atomically at the ingress layer. Built in C# / .NET, containerised, deployed by GitHub Actions.

Ingress traffic weights flip to 100% Green. In-flight Blue requests drain gracefully. Rollback is one command away.
